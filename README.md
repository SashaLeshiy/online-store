# Online Store
## Products card, Basket (add, remove), Search, Slider images, Sorting category, Payment by Cards with validation

- Vue 3
- Vite
- Typescript
- Pinia
- Fake Api
- Composition Api
- Feature-Sliced Design

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
